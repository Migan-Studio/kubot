### 실행

#### 준비

nodejs 16.6.x 이상 필수.

##### 모듈 설치

- yarn

```sh
yarn
```

##### config.json 설정

1. config.example.json 에 있는 내용을 config.json 에 복사 하세요.
2. 각 항목에 알맞게 넣어줍니다.

##### 실행

###### 컴파일 안하고 실행

```sh
yarn test
```

###### 컴파일 하고 실행

```sh
yarn build
yarn start
```
